## Getting Started

This session is dedicated to step-by-step instructions on how to use Hydra Menu.

### Prerequisites

Windows 64 Bits (idk if works on 32)
Disable Anti Virus (its not a virus, just do it if you need to)
Have WINRAR/7zip
